UX/UI Case Study

Building the VELDT Experience

A deep dive into crafting a luxury web presence that sells silence, space, and quiet confidence through code and design.

Project Overview

VELDT is a conceptual high-end fashion house inspired by raw nature, untouched earth, and organic textures. The brief was simple yet demanding: design a digital presence that feels undeniably expensive without resorting to heavy videos, flashy 3D elements, or complex WebGL. The brand language revolves around "quiet luxury" (Loro Piana meets Bottega Veneta).

The solution was an asymmetrical, single-page experience defined by massive typography, extreme negative space, and slow, intentional micro-interactions.

The Visual Language

Every element was purposefully chosen to strip away noise:

  • Color Palette: Built around `#0E0C09` (a near-black with warm undertones) rather than a rigid black, accented by `#C4A882` (muted sand/bronze).
  • Typography: We contrasted the massive, elegant *Cormorant Garamond* (light, italicized) against the severely tracked, minimalist *Montserrat* (weight 100-200) for labels.
  • Layout Philosophy: Abandoning traditional grids, cards, and hard borders allowed the content to breathe.

Technical Execution

To ensure the site felt smooth and lightweight, we relied entirely on vanilla web technologies.

  • Custom Cursor: Engineered a laggy dual-ring custom cursor using `requestAnimationFrame`, breaking the user's habitual browser rhythm to force a slower, more deliberate navigation pace.
  • Performance Native: Instead of heavy libraries like GSAP, we utilized the native `IntersectionObserver` API paired with CSS `cubic-bezier()` transitions for stagger-reveal animations.
  • Fluid Responsiveness: CSS `clamp()` functions were heavily used to scale massive display fonts perfectly from ultrawide monitors down to mobile screens.

The Outcome

The resulting landing page is a masterclass in tension and release. By providing maximum breathing room and highly choreographed, slow-paced motion, the VELDT page doesn't just display products—it evokes an emotional response of refinement and high status.
